PhotoShop application is designed for creating and manipulating images of png and jpg format.
BRUSHES
1) Pen:
Pen is designed to look like a daily pen. When the pen is applied to the canvas the digital ink from the pen will completely cover any color that had previously been applied to the canvas at that position. The color of the pen can be set interactively using the Tool Window GUI.
2) Calligraphy Pen:
Calligraphy Pen is designed to draw like an artistic calligraphy pen. The shape of Calligraphy Pen is like a vertical rectangle. It works almost the same way as the Pen tool. Like the Pen tool, it is completely opaque and it covers up any “paint” that was already applied to the canvas. The color of the Calligraphy Pen can be set interactively using the Tool Window GUI.
3) Highlighter:
Highlighter uses the same shape mask as the Calligraphy Pen, but it is semi-transparent. The darkness of the canvas color affects the strength of the applied color. For example, applying a yellow highlighter over black text will leave the text black while coloring the white canvas yellow. The color of the highlighter can be set interactively using the Tool Window GUI.
4) Spray Can:
Spray Can tool mimics the look of paint from a spray can. At the centermost position, the color is strongest and it fades as distance from center increases. The specific color to spray can be set interactively using the Tool Window GUI.
5) Eraser:
Eraser helps user erase the digital canvas and return it to its original background color.
6) Chalk:
Chalk tool mimics a chalk drawn on a board. As the user draws using this tool, a trail of chalk texture would be made on canvas. The specific color to chalk tool can be set interactively using the Tool Window GUI.
7) Rubber Stamp:
Rubber stamp takes an image loaded from a file and uses it as a stamp. The stamp is centered where the user clicks the mouse. The chosen color will affect stamp applied on canvas. For e.g. having green selected, the stamp will be greenish in shade.
8) Blur Tool:
Blur tool helps the user to blur out specific regions of canvas according to their choice. It blurs portions of canvas where ever mouse is dragged.
