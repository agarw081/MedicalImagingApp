Getting Started with MedImage:

MedImage is an application designed to be used by doctors and people associated with the medical field. It helps medical professionals edit medical scans and annotate them as per their convenience.

MedImage can be operated in two modes:

1)    GUI Mode: A graphical user interface for the application which is easy to use with additional features to annotate the image.

2)    Command Line Mode: A more simplistic and faster to use mode for the application where the user needs to type specific commands to make changes to the images.

1)  GUI Mode:

Some features of GUI mode:

1)    Multiple Images loading: The scans that doctors usually work with are numerous in count. MedImage simplifies this by the multi-image loading feature. Doctors can load multiple images at once and move between images with help of next and previous button.

 

2)    Red Pen: Red pen helps doctor to make markings on the images where ever needed. Doctors for example can mark circle on a crack in the bone with help of this tool.

 

3)    Red Stamp Marker: Red marker stamp allow doctors to stamp a red marker on the image where ever they feel important to make markings.   

 

4)    Filters: MedImage application provide doctors with different kinds of filters which help them to understand the medical scans in a better and clearer manner. The filters available to doctors are: threshold, quantize, edge detection, RGB multiply and saturate.

2)  Command Line Mode:

Commands used for command line mode are:

 

1)    Help in command line mode: medimage –h                                                                       (Brings up help menu for command line mode)

 

2)    Image conversion: medimage input.png output.jpg (PNG to JPG)

medimage input.jpg output.png (JPG to PNG)

           

3)    Edge Detection:     medimage input.png –edgedetect output.png                         (no value required)

 

4)    Threshold:             medimage input.png –threshold <float> output.png                          (float = amount of filter)

 

5)    Sharpen:                medimage input.png –sharpen <int> output.png                               (int = amount of sharpening)

 

6)    Quantize:              medimage input.png –quantize <int> output.png                              (int = amount of colors)

 

7)    Blur:                      medimage input.png –blur <float> output.png                                  (float = amount of blur)

 

8)    Saturate:                medimage input.png –saturate <float> output.png                            (float = amount of saturation)

 

9)    MultiRGB:           medimage input.png –multirgb <float><float><float> output.png   (floats = level for Red,Green,Blue)

 

10) Stamp:                   medimage input.png –stamp <int><int> output.png                         (ints = x,y coordinates for stamp position)

 

11) Compare:              medimage input.png –compare output.png                                        (returns 1 if images are identical else 0 )

 

 

Apply same filter to multiple images:  medimage input001.png –blur 2.0 output001.png

 

Apply multiple filters:                       medimage input.png –blur 2.0 –saturate 3.0 output.png
